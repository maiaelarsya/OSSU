# Introduction

- Banyak jalan menuju Roma, algoritma adalah ilmu yang mempelajari bagaimana cara menemukan jalan yang terdekat di antara banyak pilihan menuju roma. 
- Kenapa kita harus belajar tentang algoritma? karena dibanyak bidang dalam computer science, skill ini sangat dibutuhkan.
- Beberapa contoh dari penerapan algoritma:
    - komunikasi antar jaringan menggunakan algoritma shortest path
    - kriptography pada publik key menggunakan algoitma number-theoritic
    - Computer graphic menggunakan geometric algorithm
    - Search tree data structure digunakan dalam desain index database.
    - Dalam biologi misalnya, algoritma yang digunakan untuk memetakan genome adalah dynamic programming algorithm.
